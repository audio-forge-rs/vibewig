About

Voxel is an experimental project exploring AI-assisted music production. The goal is to let musicians describe what they want to hear in natural language and have an AI system translate that intent into musical parameters, controlling plugins in a digital audio workstation.

The project is built in public as a case study in human-AI collaboration for software development. Both the software and the process of building it are documented in real time.

The codebase is written in Rust, using the nih-plug framework for CLAP audio plugins. Communication between components uses Open Sound Control (OSC) over UDP. The AI assistant is Claude Opus 4.5, accessed via Claude Code CLI.
